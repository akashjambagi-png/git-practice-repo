## 📌 Core Commands Covered
Here is a quick-reference checklist of the commands I am practicing in this repository:

- [x] `git init` - Initialize a local repository
- [x] `git clone` - Clone a remote repository
- [x] `git status` - Check the state of the working directory
- [x] `git add` - Stage changes for the next commit
- [x] `git commit -m "..."` - Record changes to the repository history
- [x] `git remote add origin` - Link local repo to GitHub
- [x] `git push origin main` - Upload local commits to the remote server
- [ ] `git branch` - Manage isolation zones (branches)
- [ ] `git merge` - Combine histories together
